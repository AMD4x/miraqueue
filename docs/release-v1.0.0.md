# MiraQueue V1.0.0 Release Notes

MiraQueue V1.0.0 is the first public release.

## Highlights

- Portable PowerShell script with CMD launcher.
- Pending queue workflow for watched source changes.
- Preview and apply commands for queued changes.
- Full Mirror workflow with strict, update-keep-extras, and missing-only policies.
- Scheduled watcher support using Windows Task Scheduler.
- English documentation, maps, and simulated GIF demos.

## Known Boundaries

- Windows-only.
- Designed for local and mapped-drive style destinations.
- Full Mirror depends on `robocopy.exe`.
- V1.0.0 does not include an installer package.
